🚀 BlinkIT Sales Analytics Dashboard

Welcome to the BlinkIT Sales Dashboard 
a data analytics project designed to transform raw retail data into actionable business insights using interactive visualizations and dynamic reporting.

This project simulates a real-world quick-commerce analytics environment, where sales performance, outlet efficiency, product trends, and customer ratings are analyzed through an intuitive and visually engaging dashboard.

📊 Project Overview

The BlinkIT Sales Dashboard provides an end-to-end analytical view of retail performance by combining data analytics, business intelligence, and visualization techniques.

It enables users to:

🔍 Explore sales performance across outlet types and locations
📈 Track sales trends over time
🛒 Analyze item categories and fat content contribution
⭐ Evaluate product ratings and average sales performance
🎯 Filter insights dynamically using interactive slicers

The goal is to convert complex datasets into decision-ready insights for business stakeholders.

✨ Key Highlights
Interactive KPI Cards (Total Sales, Avg Sales, Ratings, Items)
Dynamic Filters (Item Type, Outlet Type, Outlet Size)
Sales Trend Analysis by Establishment Year
Category-wise Product Performance
Outlet Location & Size Comparison
Clean and business-focused UI design
